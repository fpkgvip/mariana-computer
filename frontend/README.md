# Mariana Computer — Frontend

React + TypeScript + Vite web application for the Mariana Computer autonomous financial research platform.

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Supabase Auth
- Stripe Payments
